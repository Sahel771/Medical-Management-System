🏥 Medical Management System

📌 Overview

This is a full-stack web-based Medical Management System developed as a university project.
The system digitalizes healthcare operations including patient management, appointment booking, prescription handling, and billing.

---

🚀 Key Features

- Patient Registration & Login
- Doctor Management System
- Appointment Scheduling
- Prescription Management
- Billing & Invoice System
- Blog & Health Tips

---

🛠 Technologies Used

- Laravel (MVC Framework)
- HTML, CSS, JavaScript
- MySQL Database
- Bootstrap
